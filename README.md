📌 WHAT THIS PACK DOES:
- Disables ALL shadows, reflections, post-processing
- Forces Performance Mode rendering
- Sets absolute minimum textures & render distance
- Enables AMD Radeon Anti-Lag & Boost
- Removes grass, fog, and all visual clutter

⚡ EXPECTED FPS BOOST: 60-120% on RX 5500 XT

📂 INSTALLATION:
1. Run "Install_All.bat" as Administrator
2. Set AMD Adrenalin settings manually (see below)
3. Launch Fortnite and verify settings

⚠️ WARNING: 
- Make backup of original GameUserSettings.ini before installing
- Game updates may reset settings - re-run installer

🔧 MANUAL AMD ADRENALIN SETTINGS:
- Radeon Anti-Lag: ON
